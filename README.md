В данной реализации однослойный перцептрон использует сигмоидальную функцию активации и функцию потерь Binary Cross-Entropy (BCE). Это делает модель математически эквивалентной логистической регрессии.

### 1. Прямой проход (Forward Pass)
Для матрицы признаков $X$ размером $N \times M$ (где $N$ — количество объектов в батче, $M$ — количество признаков) и вектора весов $w$ вычисляется линейная комбинация:
$$z = Xw + b$$
Затем к результату применяется нелинейная функция активации (Сигмоида) для получения предсказанной вероятности принадлежности объекта к классу $1$:
$$a = \sigma(z) = \frac{1}{1 + e^{-z}}$$

### 2. Функция потерь (Loss Function)
Для оценки качества предсказаний используется бинарная кросс-энтропия (BCE). Для батча из $N$ объектов она вычисляется по формуле:
$$L = -\frac{1}{N} \sum_{i=1}^{N} \left( y_i \log(a_i) + (1 - y_i) \log(1 - a_i) \right)$$
где $y_i$ — истинная метка класса ($0$ или $1$), а $a_i$ — предсказанная вероятность.

### 3. Обратное распространение (Backward Pass)
Для обучения модели необходимо найти производные (градиенты) функции потерь по весам $w$ и смещению $b$. Благодаря комбинации сигмоиды и BCE, градиенты принимают элегантный алгебраический вид:
$$dw = \frac{1}{N} X^T (a - y)$$
$$db = \frac{1}{N} \sum_{i=1}^{N} (a_i - y_i)$$

### 4. Обновление параметров (Оптимизация)
Обучение происходит с помощью классического мини-батч градиентного спуска (Mini-batch Gradient Descent). Веса корректируются в направлении, антипараллельном градиенту:
$$w = w - \eta \cdot dw$$
$$b = b - \eta \cdot db$$
где $\eta$ — гиперпараметр скорости обучения (learning rate).

## Базовый запуск модели

При стандартных параметрах обучения за 100 эпох модель демонстрирует уверенную сходимость. Функция потерь плавно снижается как на обучающей, так и на валидационной выборках, фиксируя отсутствие переобучения (overfitting).

* **Точность на обучающей выборке (Train Accuracy):** 86.32%
* **Точность на тестовой выборке (Test Accuracy):** 90.60%
